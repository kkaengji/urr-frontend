# 서비스 유저 플로우 명세

## 1. 가입 및 본인 인증

```
서비스 접속
  └─ 소셜 로그인 (카카오/네이버) 또는 이메일 가입
       └─ 본인 인증 (휴대폰 SMS, 3분 타이머)
            └─ 약관 동의 ([필수] 이용약관·개인정보 / [선택] 마케팅)
                 └─ [완료] Lv.1 미스트 즉시 부여
```

**비즈니스 로직**
- CI(연계정보) 중복 시 신규 가입 차단 → 기존 계정 안내 (1인 1계정)
- 만 14세 미만 → 가입 제한 또는 법정대리인 동의 분기
- 인증번호 만료(3분) → 재전송 버튼 활성화

---

## 2. 아티스트 멤버십 활성 & 등급 체계

```
아티스트 홈 → [멤버십 가입] → 연간 구독 결제 (₩30,000)
  └─ Lv.2 클라우드 획득 (예매/양도 권한 오픈)
       └─ 팬 신뢰 점수(FTS) 누적 → Lv.3 썬더 / Lv.4 라이트닝 승급
```

**등급 조건 & 혜택**

| 등급 | Lv | FTS 조건 | 선예매 | 예매 수수료 할인 | 양도 수수료 |
|------|----|----------|--------|-----------------|-------------|
| 라이트닝 | 4 | ≥ 85점 + Gate 통과 | T+0h (1순위) | -₩3,000 | 5% |
| 썬더 | 3 | 66–84점 | T+1h (2순위) | -₩2,000 | 5% |
| 클라우드 | 2 | 멤버십 가입 즉시 | T+1day (3순위) | -₩1,000 | 10% |
| 미스트 | 1 | 회원가입 즉시 | 잔여석만 | 없음 | 양도 불가 |

**팬 신뢰 점수(FTS) 공식**
`(티켓×0.3) + (음원×0.2) + (MD×0.15) + (플랫폼×0.15) + (신뢰×0.1) + (기간×0.1)`
- Gate 조건(예매 2회 이상, 실관람 인증률 등) 미충족 시 상위 등급 제한
- 신뢰 행동 점수 1점 미만(매크로 의심) → 즉시 미스트 강등
- 멤버십 가입 시 과거 12개월 활동 소급 계산

## 3. 등급별 차등 티켓팅

```
공연 상세 → [예매하기] → 서버 사이드 등급 검증
  └─ 대기열 진입 (순번/예상 대기시간 표시)
       └─ 구역 선택 → 좌석 선택 (7분 임시 점유)
            └─ 결제 (등급별 수수료 자동 적용)
                 └─ [완료] 마이티켓 발권
```

**비즈니스 로직**
- 등급별 오픈 시간: DB `Concert_Schedules` 기준, 서버에서 `tier_id` 검증 (클라이언트 조작 차단)
- 좌석 임시 점유 7분 → 미결제 시 즉시 해제
- 권한 없는 유저 접근 → 404 대신 "본인 오픈 시간 안내 + 멤버십 유도" 페이지

## 4. 안전 양도 및 소유권 이전

```
[판매자] 마이페이지 > 예매내역 > 양도 등록 (정가 범위 내)
  └─ 양도 게시판 리스팅

[구매자] 아티스트 홈 > 양도 게시판 (Lv.2 이상만 접근)
  └─ 양도 신청 → 안전 결제 (에스크로)
       └─ [완료] QR 재발급 + 판매자 기존 QR 즉시 무효화
```

**비즈니스 로직**
- 미스트(Lv.1) 게시판 접근 불가 (전문 리셀러 차단)
- 가격 범위: 정가의 0.5x ~ 1.5x (암표 방지 상한제)
- 소유권 이전: `ticket_owner_id` 변경 + 구 QR 폐기 + 신 QR 발급 원자적 처리
- 결제 중 동시 접근 → 선착순 `PROGRESS` 점유
- 부정 거래(단시간 다수 등록/취소) → FTS 감점 + 양도 권한 일시 정지

---

## 5. 마이페이지

**주요 기능**
- **마이멤버십**: 아티스트별 등급·FTS 점수·Gate 조건 현황
- **마이티켓**: 활성 티켓(QR 포함) / 예매·양도·취소 히스토리
- **정보 관리**: 소셜 연동 상태, 푸시 알림 설정 (선예매/양도/공지)

**비즈니스 로직**
- 티켓 공연 시간 경과 → 자동 '사용 완료' 처리 (양도/취소 버튼 비활성)
- 멤버십 만료 30일 전 → 갱신 유도 배너 + 등급 하락 경고 알림
- 탈퇴 시 CI 해시 일정 기간 보관 (재가입 시 부정 이력 확인용)
- 공연 취소 환불 → 판매자가 아닌 현재 소유자(구매자)에게 정산
